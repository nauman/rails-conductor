# Gap audit — 2026-06-20

Audit of the surfaces touched this cycle: multi-tenant MCP, kuickr publishing/slug model, the deploy pipeline, and the design system. Severity: 🔴 critical · 🟠 important · 🟡 minor. Items marked **FIXED** were resolved in this pass.

## 1. Multi-tenant MCP

| # | Finding | Sev | Status |
|---|---|---|---|
| 1.1 | `resolve_organization` had **no membership check** — a non-admin token could pass `organization_id`/`organization_slug` for any org and create_app / register_server / register_database_cluster / provision_database / set_github_token in it. | 🔴 | **FIXED** (`org_allowed?` membership gate) |
| 1.2 | `run_script` resolved servers globally (`Server.find_by`) — run a script on any org's server. | 🔴 | **FIXED** (scoped to `visible_servers`) |
| 1.3 | `set_github_app` / `github_installations` were callable by any token — instance-wide GitHub App overwrite/read. | 🔴 | **FIXED** (admin-gated) |
| 1.4 | App/server/deployment/domain/log/fleet tools looked resources up globally. | 🔴 | **FIXED** earlier (`ActorScoped`) |
| 1.5 | A token is scoped to the user's **all** orgs, not the specific `token.organization`. A multi-org user's token sees every org they belong to, not just the bound one. | 🟠 | Open |
| 1.6 | No **read-only vs deploy** token scope — any per-user token can deploy + mutate. | 🟠 | Open (slot 14) |
| 1.7 | No self-serve **token issuance/revocation UI**; tokens minted via API/console only. | 🟠 | Open (slot 14) |
| 1.8 | `ApiToken` has no **expiry/rotation**; the legacy shared `CONDUCTOR_MCP_TOKEN` is still a standing root credential. | 🟡 | Open |

**Recommendation:** 1.5–1.7 = the rest of slot 14 (bind scoping to `token.organization`; add a `scope` column read/deploy; build the issuance UI). 1.8 = add `expires_at` + a rotate flow.

## 2. kuickr publishing & slug model

| # | Finding | Sev | Status |
|---|---|---|---|
| 2.1 | **Org slugs are globally unique, first-come** (`Org.slug uniqueness: true`). Strong (no collisions) but enables **squatting**; no claim/verify/transfer flow. | 🟠 | By design — claim key slugs early (done: `conductor`, `kuickr`) |
| 2.2 | Visibility is **folder-level** only (`private`/`public`/`share_link`); no per-page visibility. Internal roadmap is private (good), but can't expose a single page. | 🟡 | By design |
| 2.3 | Private folders can't be verified except in a logged-in browser (host is session-gated; bearer/curl 404). | 🟡 | Operational — verify in browser |
| 2.4 | Orphaned `style.css` left on the kuickr `conductor/roadmap` folder after inlining. | 🟡 | Cosmetic |

**Recommendation:** if multi-tenant kuickr matters, add slug claim/transfer + a "slug taken" UX; otherwise keep first-come + a reserved list (exists).
